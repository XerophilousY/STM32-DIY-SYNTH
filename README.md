基于STM32F103C8T6温湿度计
---
系统提供多种电源输入方式，包括 Type-C 接口、3.81 mm 间距接线端子以及 DC005（5.5 mm × 2.5 mm）电源插座，以适应不同应用场景的供电需求。外部输入电压为 DC 12V。

电源管理部分采用分级降压方案：首先通过 TPS530 开关型 DC-DC 降压转换器（非线性）将输入电压降至 5V；随后由 TPS73033 或 RT9013 低压差线性稳压器（LDO）进一步稳压至 3.3V，为系统主控 MCU 及相关外设（包括传感器模块与 OLED 显示模块）提供稳定可靠的工作电源。

此外，系统设计预留标准下载接口及串口通信接口，可用于固件烧录、在线调试及数据通信，提升系统开发与维护的便利性。

1.***硬件组件***

主控：STM32F103C8T6 主频8MhZ

传感器：DHT11 单总线通信

显示方式：0.96寸OLED IIC通信


2.***开发工具***

Altium designer2020

3.***PCB版本迭代记录***

V0.0：采用的正反面设计，正面芯片，反面OLED,设计不美观，VSS和VDD连接相反

V1.0:晶振网络未连接到3.3v网络上，晶振不启振，打印出的时钟频率不对，软件倍频不生效

V2.0：使用AMS1117降压，输入12V降压至3.3V,压降太高，发热严重，电压不稳定，导致STM32死机，需要手动复位，在软件中加入看门狗后导致间隔性重启

V3.0:采用TPS5430降压至5V，后进入TPS73033降压至3.3v,没有5V输出，也没有3.3v,排查发现TPS73033输入和输出网络不一样，*导致无法形成回路*，因此没有3.3v输出

V4.0：采用四层板子设计，TPS5430用平面分割，但走线凌乱

V5.0：更改部分原件封装，删除0欧电阻作为隔离，每个部分的电源形成了回路，电源稳定

V6.0：锂电池供电版本,未单独区分

V7.0：当前版本
